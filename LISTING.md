## Plugin Name
Pen to Figma Importer

## Tagline (max 60 chars)
Import .pen design files as fully editable Figma documents

## Description
Import Pencil (.pen) design files directly into Figma — fully editable, production-ready.

Drop a JSON export of any .pen file and the plugin builds your entire design in Figma: frames with auto-layout, reusable components, instances with overrides, variable collections, fills, strokes, effects, and typography. Everything stays editable.

**What gets imported:**
- Frames with auto-layout (horizontal/vertical, gap, padding, alignment)
- Reusable components and instances with property overrides
- Variable collections from design tokens (colors, numbers, strings)
- Solid fills, linear/radial/angular gradients
- Strokes with per-side thickness and dash patterns
- Drop shadows, inner shadows, layer blurs, background blurs
- Text with font families, weights, sizes, letter spacing, line height
- Corner radius (uniform and per-corner), clipping, opacity, rotation
- Rectangles, ellipses, polygons, lines, and SVG paths

**How to use:**
1. Export your .pen file to JSON (see GitHub repo for instructions)
2. Run this plugin in any Figma file
3. Drag the .json file into the drop zone
4. Your design appears — components, pages, and all

Free and open source. Built by Vasken.
